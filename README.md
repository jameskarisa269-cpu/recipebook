# RecipeBook

A collection of text-based recipes.

## Current Recipes
- Spaghetti Carbonara
- Garden Salad
-chocolate cake

## How to Contribute
1. Fork the repository
2. Add your recipe in the recipes/ folder
3. Submit a Pull Request

## Format Guidelines
Each recipe should include:
- Title
- Ingredients (one per line with -)
- Instructions (numbered steps)
- Prep time
- Cook time
